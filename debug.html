Troubleshooting
  1. Check your Netlify environment variables: GOOGLE_CLIENT_ID and GOOGLE_API_KEY should be set correctly
  2. Ensure your Google Cloud project has the People API enabled
  3. Verify your OAuth consent screen is configured
  4. Check that your Netlify domain is added to the authorized JavaScript origins in your Google Cloud OAuth client settings
  5. Make sure the build script ran successfully
